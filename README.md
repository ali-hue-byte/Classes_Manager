# Classes_Manager


## Detailed Description
This project currently includes the initial user interface for a student management system.  
It is a work in progress, and only the login and create account screens have been implemented so far.

Features included in this prototype:

- Animated transitions between the login and create account screens
- Login screen with username and password fields
- Create account screen with username and password fields
- Smooth UI animations for widgets sliding in and out
- Placeholder for future student and class management features

Note: The backend functionality for account creation, login validation, and student management has not yet been implemented.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Classes_Manager.git
cd Classes_Manager
```
2. Install the required dependencies:

```bash
pip install PySide6
```
3. Run the main application:

```bash
python Student_Management.py
```

--- 

## Files

- **Student_Management.py** – The main Python file used to run the application.
- **Main_ui.py** – The Python file generated from Qt Designer that contains all the UI widgets and layouts.
